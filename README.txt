⚠️ The official Tuya App for Homey is currently broken for new users due to Tuya revoking API access. You will get an error saying "No matching app user information". We are working together with Tuya on fixing this issue.

Connect any Tuya device with Homey & Homey Pro.

INSTRUCTIONS
The Tuya App for Homey is compatible with both the 'Tuya Smart' and 'Smart Life' apps for iOS and Android. Before connecting your Tuya-enabled devices to Homey, ensure they are first linked to one of these apps.

CONTRIBUTE
Tuya makes many devices, and we're working together with the Homey Community to add support for all of them. If you'd like to contribute, visit the source on GitHub.
